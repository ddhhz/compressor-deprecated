Online JavaScript/CSS/HTML Compressor

This is a web interface to minify your JavaScript, CSS and HTML. This tool uses UglifyJS 2, Clean-CSS and HTML Minifier. You can easly change the defaults below the textarea.

FAQs

Why did you make this?
Not everyone is comfortable with command line interfaces or deployment pipelines. This just makes it easy if you don't have a system set up.
Do you make a copy of my code?
No. Code entered into this site is not stored anywhere. The source for this site is on GitHub at ddhhz/compressor.
